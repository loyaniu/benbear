# Project Guidelines

## Language

- All code, comments, and commit messages must be in **English only**
- No Chinese or other non-English text in source files
- Variable names, function names, and documentation should be in English

## Tech Stack

- Angular 20 + Ionic 8 + Capacitor
- Firebase (Auth, Firestore, Analytics, Vertex AI)
- Tailwind CSS v4 (utilities only, no preflight)
- Phosphor Icons

## Code Style

- Use Angular standalone components
- Follow Angular style guide conventions
- Use TypeScript strict mode

## After Coding

- Let user build it and test it out
